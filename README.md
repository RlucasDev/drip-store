# Loja Drip

Projeto de e-commerce desenvolvido para exibição de produtos, navegação por categorias e visualização detalhada de itens.

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS
- React Router DOM

## Funcionalidades

- Página inicial com produtos
- Página de categorias
- Visualização de detalhes do produto
- Login e cadastro

## Como instalar

Clone o repositório:

git clone https://github.com/RlucasDev/loja-drip.git

Entre na pasta do projeto:

cd loja-drip

Instale as dependências:

npm install

## Como executar

Para rodar localmente:

npm run dev

## Build para produção

npm run build
